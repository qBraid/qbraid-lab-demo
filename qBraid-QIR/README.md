# qBraid QIR Examples

Notebooks demonstrating [qBraid QIR](https://docs.qbraid.com/qir) features: QIR-based simulation, transpilation, and algorithm implementation.

## Notebooks

| Notebook | Description |
|----------|-------------|
| [autoqasm_deutsch_jozsa.ipynb](autoqasm_deutsch_jozsa.ipynb) | Deutsch-Jozsa algorithm implemented with AutoQASM and run on the qBraid QIR simulator |
| [qbraid_qir_sim_batch_jobs.ipynb](qbraid_qir_sim_batch_jobs.ipynb) | Batch job submission to the QIR simulator |
| [qbraid_transpiler_braket_to_qir.ipynb](qbraid_transpiler_braket_to_qir.ipynb) | Transpiling Amazon Braket circuits to QIR |
