# PyQASM Examples

Notebooks demonstrating [PyQASM](https://docs.qbraid.com/pyqasm) features: OpenQASM parsing, processing, and gate unrolling.

## Notebooks

| Notebook | Description |
|----------|-------------|
| [deutsch_josza_unroll.ipynb](deutsch_josza_unroll.ipynb) | Deutsch-Jozsa circuit gate unrolling with PyQASM |
| [qasm_processing.ipynb](qasm_processing.ipynb) | OpenQASM program processing and analysis |
| [qasm_unroll.ipynb](qasm_unroll.ipynb) | Gate decomposition and unrolling utilities |
