# qBraid Algorithms Examples

Notebooks demonstrating quantum algorithm implementations using the [qBraid Algorithms](https://docs.qbraid.com/algorithms) library.

## Notebooks

| Notebook | Description |
|----------|-------------|
| [bernstein_vazirani.ipynb](bernstein_vazirani.ipynb) | Bernstein-Vazirani algorithm implementation |
| [qft.ipynb](qft.ipynb) | Quantum Fourier Transform (QFT) |
| [qpe.ipynb](qpe.ipynb) | Quantum Phase Estimation (QPE) |
