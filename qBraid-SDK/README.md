# qBraid SDK Examples

Notebooks demonstrating core features of the [qBraid SDK](https://docs.qbraid.com/sdk) (>= 0.12.0): transpilation, device management, and visualization.

## Notebooks

| Notebook | Description |
|----------|-------------|
| [bell_circuit_examples.ipynb](bell_circuit_examples.ipynb) | Bell state circuits across 10 quantum frameworks, all run on the qBraid QIR simulator |
| [qbraid_sdk_devices.ipynb](qbraid_sdk_devices.ipynb) | Unified view of all quantum devices accessible through the QbraidProvider |
| [qbraid_sdk_transpiler.ipynb](qbraid_sdk_transpiler.ipynb) | Cross-framework circuit transpilation and equivalence verification |
| [qbraid_sdk_transpiler_braket_qiskit.ipynb](qbraid_sdk_transpiler_braket_qiskit.ipynb) | Braket-to-Qiskit transpilation examples |
| [qbraid_sdk_transpiler_cirq_stim.ipynb](qbraid_sdk_transpiler_cirq_stim.ipynb) | Custom conversion graph: Qiskit to Stim via Cirq using stimcirq |
| [qbraid_visualization.ipynb](qbraid_visualization.ipynb) | Circuit and result visualization utilities |
